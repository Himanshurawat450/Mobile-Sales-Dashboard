# 📊 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Mobile Sales Dashboard** built using **Power BI** and **Microsoft Excel**. The dashboard provides valuable insights into mobile sales performance, customer behavior, product trends, and payment preferences. It helps businesses monitor key performance indicators (KPIs) and make data-driven decisions through interactive visualizations.

---

## 🎯 Objective

The objective of this project is to analyze mobile sales data and identify:
- Overall sales performance
- Monthly sales trends
- Best-selling mobile models
- City-wise sales distribution
- Customer generation analysis
- Preferred payment methods

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

- 💰 Payment Method Analysis

---

## 📊 Key Insights

- Achieved total sales of **₹769 Million**.
- Sold over **19K mobile units** across **3,863 transactions**.
- Millennials generated the highest number of sales.
- Debit Card was the most preferred payment method.
- iPhone SE was among the top-performing mobile models.
- Sales were distributed across multiple cities in India.
